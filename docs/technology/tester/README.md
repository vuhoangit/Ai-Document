# Tester

Use this folder for software testing knowledge such as:

- Testing principles, roles, and quality mindset
- Test planning, estimation, and risk-based testing
- Manual and exploratory testing workflows
- API/UI automation and test framework strategy
- Non-functional testing (performance, security, reliability)
- Defect lifecycle, reporting quality, and triage process

- `testing-fundamentals`
- `test-design-techniques`
- `test-management-and-planning`
- `manual-testing`
- `api-testing`
- `ui-testing`
- `test-automation`
- `performance-testing`
- `security-testing`
- `mobile-testing`
- `defect-management-and-reporting`
